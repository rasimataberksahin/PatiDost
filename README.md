# 🐾 PatiDost Veterinary Management System

PatiDost is a veterinary clinic management system developed using ASP.NET Core Web API.

## Features

* Pet Owner Management
* Animal Records
* Appointment Scheduling
* JWT Authentication
* Feature-based Authorization

## Technologies

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* JavaScript Frontend

## Purpose

This project was developed as a real-world backend practice application.
